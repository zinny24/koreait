1. 설치
2. 깃 환경 설정하기
   - git config --global user.name "이름"
   - git config --global user.email "메일주소"
3. pwd, ls, ls -al, ls -alr 거꾸로 정렬
   ls -alt 시간 거꾸로
   clear
   cd
   cd /d/koreait
   mkdir
   rm -r


4. 파일 또는 폴더 만들기
  - 폴더와 깃 연결하기 git init
  - ls -al로 .git 숨김 파일 확인하기

5. git status
  - untracked : 한번도 버전 관리하지 않은 파일이 있습니다.
  - git add database 버전관리를 할 거다. 하지만 아직 업로드는 아니다.

6. 이제부터 버전관리한다(업로드)
  - git commit -m "Koreait Tutorial Data"
  - git status
  - git log : 전체 정보 확인하기

7. 회원가입 후 화면 오른쪽 위 더하기 버튼 - New Repository - 이동 - 주소확인

8. git remote add orgin 깃 주소
   - git remote -v : 연결 확인

9. 로컬 저장소와 깃 서버 저장소 연동하기
 - git branch -M main
 - git push -u origin main : 파일 전송(단 처음일 때는 아래 인증 후 전송)

 - 로그인 창이 열림 : Sign your browser
   Authorize 선택
   사용자 인증 완료하기
