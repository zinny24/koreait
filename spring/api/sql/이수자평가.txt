[직무관련부분]
1. JSON이 무엇인가?
- 서버에서 클라이언트로 데이터를 보낼 때 사용하는 양식으로 자바스크립트 객체의 형식을 기반으로 만들어졌다.
- 자바스크립트의 문법과 굉장히 유사하지만 텍스트 형식일 뿐이다.

2. DDL이 무엇인지 정의하라?
- 데이터베이스 정의어를 말하며, 데이터를 생성, 수정, 제거하는 명령어를 말한다.
- CREATE(생성), ALTER(수정), DROP(제거), TRUNCATE(테이블 비우기-초기화)

3. string과 int의 차이는 무엇인가 ?
- 문자(열)형, 정수형

4. 평가에서 사용한 코드는 무엇이고, 그 코드에 대해 설명하라?
- JAVA, HTML, CSS, JAVASCRIPT, MYSQL, MYBATIS, THYMELEAF...

6. git이 무엇인가?
- 소스코드를 여러 개발 PC와 저장소에 분산해서 저장/관리하는 형상관리(버전관리)도구이다.

7. 훈련내용에 인덱스가 포함 되어 있었는가. 그렇다면 인덱스에 대해서 설명하라
- 인덱스는 데이터베이스 테이블에 대한 검색 성능의 속도를 높여주는 자료구조이다. 
- 특정 컬럼에 인덱스를 생성하면, 해당 컬럼의 데이터들을 정렬하고 별도의 메모리 공간에 저장하여 처리하여 검색 성능을 높인다.

8. 자바 스프링에 대하여 아는 대로 설명하라
- JAVA로 다양한 웹 어플리케이션을 만들기 위한 프레임워크이다. 프레임워크란 규칙을 정해 놓고 개발하는 틀과 같은 것을 말한다.

9. MVC 모델 작동법은 무엇인가
- 각 기능을 분리하여 개발 및 유지 보수가 편리하게 하기 위함.
- Controller : 사용자 요청 처리 및 흐름 제어를 담당(URL 처리 등)
- Model : 비즈니스 로직 처리(디비에 저장)
- View : 사용자에게 보여줄 화면  담당

11. 스프링 프레임워크의 뷰(View)
- Thymeleaf는 HTML, XML, JavaScript, CSS 및 일반 텍스트를 처리 할 수 있는 웹 및 독립형 환경에서 사용할 수 있는 Java 템플릿 엔진입니다. 
  Thymeleaf는 자바코드를 사용할 수 없고, jsp에서 처럼 커스텀 태그와 같은 기능도 없습니다.
- JSP는 서블릿으로 변환되어 실행이 되어지며 JSP 내에서 자바 코드를 사용할 수도 있습니다. 


13. 추상 메서드가 무엇인가?
- 여러 객체에 공통되는 사항을 추출하여 부모 클래스에 선언하고 자식 클래스에는 특징을 구현하여 기능을 확장하는 것을 말한다.
  또 추상 메서드를 사용함으로서 개발을 표준화하여 진행할 수 있다.
- 메서드 앞에 abstract 키워드를 사용하며 몸통(중괄호)은 없고 타입, 이름만 존재한다.


14. DML은 무엇인가?
- 데이터베이스 조작어
- 데이터베이스에 입력된 레코드를 입력, 조회, 수정, 삭제하는 질의어를 말한다.
- SELECT, INSERT, UPDATE, DELETE

15. SQL의 특성
- SQL은 Structured Query Language의 줄임말로, 관계형 데이터베이스 시스템(RDBMS)에서 데이터를 관리, 처리하기 위해 설계된 프로그래밍 언어이다. 
- DDL(정의어), DML(조작어), DCL(제어어)가 있다.

16. 자바를 사용할 때 접근 제어 방법은?
1) private
   - private 접근 제어자를 사용하여 선언된 클래스 멤버는 외부에 공개되지 않으며, 외부에서는 직접 접근할 수 없습니다.
2) public
   - public 접근 제어자를 사용하여 선언된 클래스 멤버는 외부로 공개되며, 해당 객체를 사용하는 프로그램 어디에서나 직접 접근할 수 있습니다.
3) default
   - 접근 제어자가 지정되지 않으면 자동적으로 default 접근 제어를 가지게 됩니다.
   - 같은 클래스의 멤버와 같은 패키지에 속하는 멤버에서만 접근할 수 있습니다.
4) protected
   - protected 멤버는 부모 클래스에 대해서는 public 멤버처럼 취급되며, 외부에서는 private 멤버처럼 취급됩니다.

19. 오버로딩의 개념에 대해 설명하라
   - 메소드의 이름이 같고, 매개변수의 개수나 타입이 달라야 한다. 주의할 점은 '리턴 값만' 다른 것은 오버로딩을 할 수 없다는 것이다.

20. UI디자인에서 테이블 구성에는 어떤 것들이 있는가?
- table, thead, tbody, tfoot, tr, th, td, caption


[능력획득여부]
1. SELECT
- SELECT * FROM 테이블명;
- SELECT 컬럼명1, 컬럼명2, 컬럼명3 FROM 테이블명;
- SELECT * FROM 테이블명 ORDER BY 정렬할 컬럼명 DESC(ASC);
- SELECT COUNT(컬럼명) FROM 테이블명;
- SELECT MAX(컬럼명) FROM 테이블명;
- SELECT SUM(컬럼명) FROM 테이블명;
- SELECT COUNT(컬럼명) FROM 테이블명 Limit 개수;
- SELECT COUNT(컬럼명) FROM 테이블명 Limit 시작번호, 개수;
- SELECT 그룹처리할 컬럼명, 집계함수 FROM 테이블명 GROUP BY 그룹처리할 컬럼명;
  ex) 부서별 급여의 합
       - SELECT 부서컬럼,  SUM(급여컬럼) FROM 테이블명 GROUP BY 부서컬럼;
  ex) 등록된 제조사별 차량의 수
       - SELECT company, COUNT(*) FROM 테이블명 GROUP BY  company;

- SELECT * FROM 테이블명 WHERE 찾을컬럼명 = 값;
  ex) SELECT * FROM 테이블명 WHERE id = 1;

- 유사어 검색
  - SELECT * FROM 테이블명 WHERE 컬럼명 LIKE '%검색어%';
    ex) SELECT * FROM 테이블명 WHERE name LIKE '%홍%'; -- 이름에 홍이 포함된 사람 모두 찾기
     


2. INSERT
- INSERT INTO 테이블명 VALUES(값1, 값2, 값3...);

3. UPDATE
- UPDATE 테이블명 SET 변경할컬럼1=값, 변경할컬럼2=값 WHERE 바꿀컬럼명 = 값;
  ex) UPDATE 테이블명 SET name='홍길동', passwd='1234' WHERE name = '부산광역시';

4. DELETE
- DELETE FROM 테이블명 WHERE 컬럼명 = 값;
  ex) DELETE FROM 테이블명 WHERE userid = 'busan';
