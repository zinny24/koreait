16. 자바를 사용할 때 접근 제어 방법은?
1) private
   - private 접근 제어자를 사용하여 선언된 클래스 멤버는 외부에 공개되지 않으며, 외부에서는 직접 접근할 수 없습니다.
2) public
   - public 접근 제어자를 사용하여 선언된 클래스 멤버는 외부로 공개되며, 해당 객체를 사용하는 프로그램 어디에서나 직접 접근할 수 있습니다.
3) default
   - 접근 제어자가 지정되지 않으면 자동적으로 default 접근 제어를 가지게 됩니다.
   - 같은 클래스의 멤버와 같은 패키지에 속하는 멤버에서만 접근할 수 있습니다.
4) protected
   - protected 멤버는 부모 클래스에 대해서는 public 멤버처럼 취급되며, 외부에서는 private 멤버처럼 취급됩니다.

[능력획득여부]
1. SELECT
- SELECT * FROM 테이블명;
- SELECT 컬럼명1, 컬럼명2, 컬럼명3 FROM 테이블명;
- SELECT * FROM 테이블명 ORDER BY 정렬할 컬럼명 DESC(ASC);
- SELECT COUNT(컬럼명) FROM 테이블명;
- SELECT MAX(컬럼명) FROM 테이블명;
- SELECT SUM(컬럼명) FROM 테이블명;
- SELECT COUNT(컬럼명) FROM 테이블명 Limit 개수;
- SELECT COUNT(컬럼명) FROM 테이블명 Limit 시작번호, 개수;
- SELECT 그룹처리할 컬럼명, 집계함수 FROM 테이블명 GROUP BY 그룹처리할 컬럼명;
  ex) 부서별 급여의 합
       - SELECT 부서컬럼,  SUM(급여컬럼) FROM 테이블명 GROUP BY 부서컬럼;
  ex) 등록된 제조사별 차량의 수
	집계함수 : sum, avg, max, min, count
       - SELECT company, COUNT(*) FROM 테이블명 GROUP BY  company;

- SELECT * FROM 테이블명 WHERE 찾을컬럼명 = 값;
  ex) SELECT * FROM 테이블명 WHERE id = 1;

- 유사어 검색
  - SELECT * FROM 테이블명 WHERE 컬럼명 LIKE '%검색어%';
    ex) SELECT * FROM 테이블명 WHERE name LIKE '%홍%'; -- 이름에 홍이 포함된 사람 모두 찾기
